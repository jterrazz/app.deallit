# mymarket-server

> A marketplace with decentralized applications

## Requirements
- MySQL
- Redis
- ZeroMQ
- Bitcoind (bitcoin-core)

## Build Setup

``` bash
# install dependencies
npm install

# command for development
npm run dev

# command for production
npm run prod

# cluster mode for development
npm run pm2-dev

# cluster mode for production
npm run pm2-prod
```
